Repositório com programa em C, de busca de Logradouro ou Cep através de um Logradouro ou Cep.
Utilizando busca por Arvore B, em um arquivo de texto binário onde cada linha é referente a um Registro.
Formato do registro: 
!(CEP sem -);(Logradouro)           ---complemento em espaços até 90º caracter---       .
